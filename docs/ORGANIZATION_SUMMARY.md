## 🚀 Usage Guidelines

### Adding New Files

#### Documentation

```bash
# Add to docs/ folder
docs/NEW_FEATURE_GUIDE.md
docs/API_DOCUMENTATION.md
```

#### Tests

```bash
# Add to tests/ folder
tests/test-new-feature.js
tests/test-api-endpoints.js
```

#### Scripts

```bash
# Add to scripts/ folder
scripts/database-backup.js
scripts/cleanup-logs.js
```

#### Temporary Files

```bash
# Use temp/ folder (automatically gitignored)
temp/debug-output.log
temp/test-data.json
```

### Finding Files

#### Need Documentation?

- **Location**: `docs/` folder
- **Main Guide**: `README.md`
- **Structure Info**: `docs/PROJECT_STRUCTURE.md`

#### Need to Run Tests?

- **Location**: `tests/` folder
- **System Test**: `tests/test-system.js`
- **Backend Test**: `tests/test-backend.js`

#### Need Setup Scripts?

- **Location**: `scripts/` folder
- **Environment Setup**: `scripts/setup-env.js`

#### Need Deployment Info?

- **Location**: `deployment/` folder
- **cPanel Deploy**: `deployment/deploy-cpanel.js`
